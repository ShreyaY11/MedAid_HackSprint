# MedAid: AI-Powered Early Disease Detection System

MedAid is a Django-based web platform that uses state-of-the-art AI (OpenAI/Gemini) to predict early signs of critical diseases such as diabetes, cardiovascular conditions, and cancer. Designed for both healthcare providers and patients, MedAid is your assistant for early, reliable, and actionable health risk assessment.

- **Prediction Dashboard:** Color-coded risk, AI explanations, and confidence levels.
- **Patient Management:** Register, search, and manage patients from a modern admin dashboard.
- **Secure Auth:** Password security, session management, CSRF protection.
- **Responsive UI:** Mobile-friendly and visually appealing (Bootstrap 5, custom theming).
- **Data Privacy:** Local (not cloud) storage for all patient data.
